# Propuesta para Antony Fulgencio Real Estate

## Objetivo

Crear una presencia digital premium que no dependa de publicar disponibilidad exacta, sino de demostrar confianza, entregas, proceso y capturar clientes calificados.

## Fase 1: Pagina de autoridad y captacion

Incluye:

- Diseno premium con marca de Antony.
- Logo optimizado sin fondo.
- Perfil profesional.
- Secciones de confianza, zonas, compras desde el exterior y evaluacion del caso.
- Botones de WhatsApp.
- Formulario que prepara mensaje para contacto.
- Publicacion en GitHub Pages.
- SEO basico.

Precio sugerido:

- RD$45,000 a RD$75,000 si se entrega como landing premium sin backend.
- US$800 a US$1,500 como rango internacional razonable para este alcance.

## Fase 2: Contenido real y conversion

Incluye:

- Sustituir imagenes genericas por fotos reales.
- Subir casos de entregas.
- Preparar textos de testimonios.
- Insertar videos o reels autorizados.
- Configurar numero real de WhatsApp.
- Optimizar mobile y mensajes de venta.

Precio sugerido:

- RD$25,000 a RD$60,000 adicional, dependiendo de cuantas fotos, videos y cambios traiga el cliente.

## Fase 3: Plataforma real con administracion

Incluye:

- Panel administrativo con login.
- Base de datos.
- Subida permanente de fotos/videos.
- Gestion de casos, zonas, testimonios y leads.
- Formularios guardados.
- Analytics.
- Dominio propio.

Precio sugerido:

- RD$120,000 a RD$250,000+ segun alcance.
- US$2,500 a US$5,000+ si incluye backend, storage, panel y configuraciones.

## Mantenimiento mensual

Incluye:

- Cambios pequenos.
- Subida de nuevos casos.
- Ajustes de textos.
- Revision de enlaces.
- Soporte basico.

Precio sugerido:

- RD$8,000 a RD$20,000 mensual.
- US$150 a US$350 mensual.

## Recomendacion comercial

Para la reunion, venderlo por fases:

1. Lanzar pagina premium y captacion.
2. Cargar contenido real de entregas y testimonios.
3. Convertirlo en plataforma con panel y base de datos.

No vender disponibilidad en tiempo real a menos que exista un sistema real para mantenerla actualizada.
