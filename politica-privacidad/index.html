Politica de privacidad

La informacion enviada por formularios o WhatsApp se usa para responder solicitudes de asesoria inmobiliaria, coordinar seguimiento y compartir opciones relacionadas con el interes del cliente.

No se vende informacion personal a terceros. Los datos pueden compartirse solamente cuando sea necesario para gestionar una orientacion, visita, validacion de disponibilidad o proceso inmobiliario solicitado por el cliente.

Para solicitar actualizacion o eliminacion de datos, escribe a antony@tucasard.com.
